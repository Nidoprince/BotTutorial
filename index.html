Making a Discord Bot with Node.js

1. Make a Discord Server

If you already have a server you want to host your bot on, then you can skip this step, but otherwise, within Discord, click on the + button bellow the server list and make a new server.

2. Make Application

While logged into Discord, go to: https://discordapp.com/developers/applications/.

Click the "Create an App" button. Then in settings, click "bot". Click "Add Bot".

3. Record ID and Bot Token

In the general information setting, you can find the id. Copy this down. Then go to the bot section, and copy the bot token.

4. Add the Bot to your Server

Copy the following url: https://discordapp.com/oauth2/authorize?&client_id=CLIENTID&scope=bot&permissions=0

Now, replace the part that says CLIENTID with the id you recorded in the previous step. Paste this into your browser and go there. Then select the server you want to add the bot to and click authorize.

5. Write the package.json file

Make a folder where you want to keep the bot on your computer. In that folder you want to make a new file called package.json. Look at the code in this link to see what you should put in it, changing the author and the bot name to whatever you want.

6. Write the Code

We are using javascript for this tutorial, so make a file called testBot.js, or whatever name you want for it. If you choose a different name though, make sure to change the name in the package.json file as well. Again, you can follow the exapmple in this link.

7. Install Packages

Open up a console, or powershell, naviage to the folder you have your code in, and then type npm install. You'll get some warnings, but you can ignore them, as long as the discord.js package is installed. You can check that it did install by looking in the node_modules folder.

8. Save token

Now, we need to save our bot token in a way that will not be shared if we share the code. There are a couple ways to do this, but the way it works in the example code is that you need to save the token in your environment variables.

Search your computer for environment variables. It should open up something called "system properties". Click "Environment Variables" at the bottom of the Advanced tab. In the lower half of the dialogue that pops up, click new. Name the vairable TEST_BOT_TOKEN and make the value the token you copied down in step 3. Hit OK.

9. Run the bot

Open a new console or powershell instance. The one you had open before will not have the new environment variable you just set. Navigate to your bot folder and type: npm start

After a moment you should see a few things, and it will say "I am ready!". If so, your bot is running. You can go into Discord and type !hello and it will respond. If there is an error, then one of the steps got messed up and you will have to debug.

10. Add additional functionality

Now that you have the basic functionality working, it's not too hard to add more to the bot. We are using the discord.js api, so you can check out their documentation at: https://discord.js.org/#/. Look around there, and start making something cool.
